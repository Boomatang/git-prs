## ADDED Requirements

### Requirement: Project README
The project SHALL have a README.md in the repository root.

#### Scenario: README contains essential sections
- **WHEN** a user views the repository
- **THEN** README includes: overview, prerequisites, installation, quick start, and links to detailed docs

### Requirement: CLI reference documentation
The project SHALL have detailed CLI documentation in docs/cli-reference.md.

#### Scenario: CLI reference documents all commands
- **WHEN** a user reads the CLI reference
- **THEN** they find documentation for: mine command, team command, all flags, and usage examples

### Requirement: Configuration guide
The project SHALL have a configuration guide in docs/config-guide.md.

#### Scenario: Config guide explains file format
- **WHEN** a user reads the config guide
- **THEN** they find: config file location, JSON schema, minimal example, full example, and troubleshooting
